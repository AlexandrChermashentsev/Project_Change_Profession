Задача: Написать программу, которая из имеющегося массива строк формирует
массив из строк, длина которых меньше, либо равна 3 символа. 
Первоначальный массив можно ввести с клавиатуры,
либо задать на старте выполнения алгоритма.
При решении не рекомендуется пользоваться коллекциями,
лучше обойтись исключительно массивами.
Примеры: 
[«Hello», «2», «world», «:-)»] -> [«2», «:-)»]
[1234, 1267, -2, computer science] -> [-2]
[Russia, Denmark, Kazan] -> []

Решение:
Сначала пользователь вводит длину первоначального массива(защиту от дурака,
к сожалению, я не очень понял как делать, поэтому если пользоваетль вместо положительного числа,
которое его просят ввести, введет что то другое, то программа крашится).

Далее пользоваетель вводит вручную значение каждого элемента массива userArray. 
За это отвечает функция InitialArray
Тут у него полное раздолье, что хочет, пусть то и вводит(даже может ничего не вводить)
Затем выводим, в консоль, первоначальный массив(выводим даже пустое значение элементов в виде <null>)

Далее находим наш новый массив с элементами <=3 символам. За это отвечает функция FindThreeElementsArray.
В этой функции мы сначала определяем длину финального массива, чтобы правильно его инициализировать. Пробегаемся по каждому элементу 
пользовательского массива и проверяем условие "Элемент массива <= 3 символам" и ячейка массива не должна быть пустой.
Затем, в этой же функции мы снова проходимся по нашему пользоваетельскому массиву и кладем каждый елемент <= 3 символами
в новый фианльный массив. Этот массив мы и получаем на фыходе из функции.

Печатем полученный массив.

P.S. При решении я не учитывал загруженность компьютера(когда дважды пробегался по одному и тому же массиву, с одним и тем же условием)
Думаю есть куда более простое решение, но пока при тех знаниях и опыте, которые у меня есть, это решение мне кажется оптивальным.
Так же я не стал сильно заморачиваться над красотой вывода масивов, так при некотрых обстоятельствах могут вылазить ненужные запятые и пробелы,
но меня же не за красоту вывода программы сегодня оценивают? :-)
